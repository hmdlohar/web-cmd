# Node Tunnel

A simple tunneling service similar to ngrok, allowing you to expose local services to the internet.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/node-tunnel.git
   cd node-tunnel
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
